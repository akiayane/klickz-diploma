Klic.kz API

В Klickz API v1.0.1 на данный момент доступен следующий функционал.

  1. Создание ссылки
  2. Получение данных по ссылке
  3. Получение метаданных по IP

Для создания ссылки необходимо отправить POST запрос

В тело запроса необходимо передать - address ( Адрес ссылки которую необходимо обработать )

В Заголовки запроса необходимо передать ваш Klickz-api-token
